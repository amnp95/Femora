## Femora API Reference

Femora is a next-generation Python library designed to simplify and accelerate the creation of Finite Element Method (FEM) models for **OpenSees**.

This section documents the Python API for:

- **Meshing**: `femora.meshmaker`
- **Materials**: `femora.material`
- **Analysis**: `femora.analysis`
